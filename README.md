# OpenCV 3.2.0

运行前请先安装[OpenCV Manager(必须)](https://github.com/kongqw/FaceDetectLibrary/tree/opencv3.2.0/OpenCVManager)。

本示例基于OpenCV 3.2.0 版本在Android平台实现目标检测和目标追踪

## 目标检测

- 人脸检测
- 眼睛检测
- 微笑检测
- 上半身检测
- 下半身检测
- 全身检测

## 目标追踪

- CamShift算法实现目标追踪
